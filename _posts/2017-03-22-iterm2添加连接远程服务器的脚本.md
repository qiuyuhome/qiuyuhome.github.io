重点是箭头指向的地方:

填入的格式:
`expect /Users/qiuyu/iterm_login_shell/192.168.10.2`

其中
`/Users/qiuyu/iterm_login_shell/192.168.10.2`是我的文件路径. 如图:

![-w400](/images/posts/14901484425066.jpg)

内容如下:

```
#!/usr/bin/expect -f
set user root
set host 192.168.10.2
set password 111111
set timeout -1

spawn ssh $user@$host
expect "*assword:*"
send "$password\r"
interact
expect eof
```

这样, 以后按`command+o`, 就可以呼出选项了.
